# Agencia-Ruta-7

Sitio web estático creado para el dominio `agenciaruta7.cl`.

## Archivos principales

- `index.html` — página principal del sitio.
- `css/style.css` — estilos responsive y diseño moderno.
- `js/main.js` — scripts de interacción para el menú móvil y año automático.
- `CNAME` — nombre de dominio para despliegue en GitHub Pages o como referencia de hosting.

## Características incluidas

- Presentación de servicios: desarrollo web, SEO, Meta Ads, seguridad.
- Fases de trabajo: análisis, ideas, desarrollo, optimización.
- Portafolio con sitios clave: ChileCompra, Trebol.IT, San Luis, PortatilChile, Dumay.
- Contacto directo por WhatsApp y correo electrónico.
- SEO básico con meta tags y schema.org.
- Diseño dinámico, profesional y responsivo.
